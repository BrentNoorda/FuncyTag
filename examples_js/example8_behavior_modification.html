example8_behavior_modification - special options to modify tag behavior (_nobrout, _nobrin , _noslfcls, no options)

  • _nobrout:true - Turn off newlines (and all whitespace) around this tag; default false
  • _nobrin:true - Turn off newlines (and all whitespace) inside this tag; default false
  • _noslfcls:true - Do not allow tag to be self-closing (forces <tag></tag> instead of <tag/>); default false
  • no options - if no options are given (i.e., the first paramer is not a plain object) then {} is assumed
