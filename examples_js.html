FuncyTag

FuncyTag is a way to generate HTML in pure javascript, and make it look real nice and javascripty.

Learn a bunch more, including the python library, at FuncyTag on GitHub.

FuncyTag Javascript Examples

  • example1_intro - demonstrate very basic use of Tags as Functions
  • example2_children - a little more
  • example3_css_styles - treat css styles as tag attributes
  • example4_css_units - add units to the end of attribute and css names
  • example5_inner_tag - contents of a tag may be another tag
  • example6_array_contents - contents of a tag may include arrays of content
  • example7_default_attributes - create new tags with default attributes
  • example8_behavior_modification - options to modify tag behavior
  • example9_inheritance - tags can inherit options of other tags
  • example10_esc - escaping text within a string
  • example_end - no more examples
For more info, see FuncyTag on GitHub
