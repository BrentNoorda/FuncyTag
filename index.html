FuncyTag & FuncyStyle

Generate HTML/JS/CSS into nested Javascript functions.
FuncyTag & FuncyStyle allow all components of a web page (HTML, CSS, and Javascript) to be generated from purely functional source that still looks pretty much like standard HTML/CSS/JS. Easy to read, easy to write, easy to reuse. Source-code locality.

What is this?

This is an experiment to answer the questions: What if web pages could be developed purely in JavaScript? Under what circumstances does that make development easier?

What do FuncyTag & FuncyStyle do?

FuncyTag & FuncyStyle generate HTML and CSS and Javascript event handlers and everything else a browser for presentation, directly from nested, functional javascript.

Why? What's wrong with traditional HTML+CSS+JS?

HTML+CSS+JS are great and are the go-to solution for most circumstances. But there are times when thinking in three different languages, and juggling three different source files gets really annoying. At those times, it can be really convenient to combine all of those components into a single Javascript function call that acts like a tag with it's own styling and event behaviors.
